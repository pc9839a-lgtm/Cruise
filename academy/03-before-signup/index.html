BEFORE SIGNUP

가입 전 네 가지를
정확히 알아야 합니다.

01시작 비용가입할 때 한 번 결제되는 금액
02월 납부액달러로 매월 자동 결제되는 금액
03환불·해지환불 가능 여부와 해지 후 변화
04유지 조건예약 후 탑승 전까지 확인할 상태
CLASSIC EXAMPLE

월 납부액만 보면
첫해 비용을 놓칩니다.

시작 비용$200가입 시 한 번 결제
+
월 납부$1,200$100 × 12개월
=
첫해 결제$1,400환율·카드 수수료 별도

가입 리워드와 적립 포인트는 결제 금액과 별도로 확인합니다.

DOLLAR BILLING

같은 플랜도 원화 청구액은
매달 달라질 수 있습니다.

RATE결제일 환율달러 금액이 같아도 원화 금액은 달라집니다.
FEE해외결제 수수료카드사별 해외 이용 수수료를 확인합니다.
LIMIT카드 한도정기 결제일 전 해외결제 한도를 확인합니다.
DATE정기 결제일결제 예정일과 카드 유효기간을 확인합니다.
REFUND vs CANCEL

환불과 해지는
같은 말이 아닙니다.

REFUND이미 결제한 금액
  • 환불 가능 기간 확인
  • 사용 이력과 신청 시점 확인
  • 공식 안내가 최종 기준
CANCEL앞으로의 월 결제
  • 이후 자동결제 중단
  • 추가 적립분 변화 가능
  • 예약 상태는 별도 확인
14 DAYS

가입 후 14일은
반드시 따로 확인합니다.

신청 시점가입일과 환불 신청일 확인
14일 이후환불이 제한될 수 있음
사용 이력포인트·예약 사용 여부 확인
공식 규정가입 당시 안내를 우선 적용
CANCEL EFFECT

해지하면 추가 적립분이
줄어들 수 있습니다.

클래식 12개월 월 적립2,400P가입 리워드 제외
→
해지 후 예시1,200P기본 적립분 기준
·
현금 출금불가예약에 사용하는 포인트

실제 잔여 포인트는 계정의 포인트 구분과 해지 당시 규정을 확인합니다.

POINT RULE

포인트는 현금이나
원금 보장 상품이 아닙니다.

WITHDRAW현금 출금 불가계좌로 인출하는 금액이 아닙니다.
USE여행 예약 사용크루즈·호텔 등 예약에 사용합니다.
VALUE예약 조건 확인상품과 출발일에 따라 사용 범위가 다릅니다.
RESERVATION STATUS

멤버십 해지와 예약 취소는
따로 확인해야 합니다.

멤버십 해지월 자동결제와 포인트 상태추가 적립분과 계정 상태를 확인합니다.
여행 예약 취소선사·호텔의 취소 규정예약별 취소료와 환불 가능 금액을 확인합니다.
PAYMENT CARD

정기 결제에 사용할 카드를
가입 전에 확인합니다.

OWNER본인 명의회원 정보와 카드 명의를 맞춥니다.
OVERSEAS해외결제 허용해외결제 차단 여부를 확인합니다.
EXPIRY유효기간예약일까지 사용할 수 있는지 확인합니다.
LIMIT결제 한도정기 결제가 실패하지 않도록 관리합니다.
PAYMENT FAILURE

월 결제가 실패하면
계정 상태가 달라질 수 있습니다.

적립 중단해당 월 포인트 미적립 가능
상태 변경멤버십 상태 확인 필요
예약 영향탑승 전 유지 조건 확인
카드 재등록한도·유효기간·차단 확인
KEEP ACTIVE

예약이 끝났다고
확인이 끝나는 것은 아닙니다.

01
예약 완료상품별 멤버십 유지 조건을 확인합니다.
02
월 결제 확인정기 결제가 정상 승인됐는지 확인합니다.
03
출항 전 점검계정 상태와 예약 상태를 다시 확인합니다.
04
탑승 준비여권·항공·숙박·별도 비용을 준비합니다.
