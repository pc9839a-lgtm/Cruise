INITIAL PAYMENT

첫 결제액에는 활성화 비용과
첫 달 결제가 함께 포함됩니다.

STARTER총 $50활성화 $0 + 첫 달 $50
CLASSIC총 $200활성화 $100 + 첫 달 $100
PREMIUM총 $500활성화 $250 + 첫 달 $250

이후 월 결제액은 STARTER $50 · CLASSIC $100 · PREMIUM $250입니다.

30일을 넘기면 베이스로 전환됩니다.

예약 기능 사용즉시 가능

15일 대기기간은 적용하지 않습니다. 로그인 후 크루즈·호텔 검색과 예약을 바로 진행할 수 있습니다.

14-DAY REFUND LIMIT

환불 요청일 기준 14일을 넘긴 결제는
환불받을 수 없습니다.

14일 이내 결제환불 자격 심사 대상
14일 초과 결제환불 불가
이전 월 결제소급 환불 불가
반환 수수료발생 비용은 환불액에서 차감

거주 국가·지역의 법정 취소기간이 더 긴 경우에만 해당 법정 규정이 우선합니다.

REFUND RESULT

환불을 요청하면 계정은 폐쇄되고
Reward Points를 전부 잃습니다.

01계정 영구 폐쇄환불 요청으로 전체 계정 종료
02포인트 전부 상실사용 가능한 Reward Points 소멸
03대상 결제만 환불14일 이내 환불 자격 결제만 반환
04지불 거절도 동일분쟁·차지백 시 계정 영구 폐쇄
FINAL SALE

크루즈 예약은 최종 판매입니다.
어떠한 개인 사유라도 환불되지 않습니다.

개인 사유 취소결제금 환불 불가
일정 변경 요청결제금 환불 불가
예약자 이름 변경다른 사람으로 변경 불가
호텔·리조트 예약동일하게 최종 판매

예약 취소를 요청하더라도 결제금은 반환되지 않습니다.

월 결제를 30일 넘게 미납하면
유료 멤버 자격이 끝납니다.

ACCESS베이스 전환INsider Pricing과 유료 혜택 중단
POINTSBase Points 재계산순 납부액 기준 포인트만 유지
REWARD추가 적립분 소멸두 배 적립·무료 인센티브분 제거
REACTIVATION

베이스 상태에서 다시 시작하려면
CLASSIC 또는 PREMIUM으로 재활성화해야 합니다.

재활성화 결제총 초기 결제액 다시 납부
  • CLASSIC 또는 PREMIUM 선택
  • INsider Pricing과 Reward Points 적립 재개
  • 재활성화 완료 즉시 예약 가능
제한사항STARTER 재활성화 불가
  • STARTER로 다시 활성화할 수 없음
  • Base Points는 Reward Points로 복원되지 않음
  • 기존 Base Points는 별도로 사용
STAY ACTIVE

Reward Points로 예약했다면
승선 수속 완료까지 유료 상태를 유지해야 합니다.

01월 결제 계속예약 후에도 유료 멤버 자격 유지
02대기·베이스 전환inCruises가 예약을 취소할 수 있음
03사용 포인트예약 취소 시 환불 불가
04본인 부담금예약 취소 시 환불 불가
FINAL CHECK

아래 조건을 모두 이해한 뒤
가입을 결정합니다.

  • 01가입과 재활성화 완료 직후 바로 예약할 수 있습니다.
  • 02환불 요청일 기준 14일을 넘긴 결제는 환불 불가입니다.
  • 03환불 요청 시 계정이 영구 폐쇄되고 포인트를 전부 잃습니다.
  • 04크루즈·호텔 예약은 최종 판매이며 환불·이름 변경이 불가합니다.
  • 0530일 미납 시 베이스로 전환되고 추가 적립분이 사라집니다.
  • 06포인트 예약 후 체크인까지 유료 상태를 유지해야 합니다.
